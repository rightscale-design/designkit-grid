# Designkit Grid

Sass module for simple css grids.

## Install

```bash
npm i designkit-grid
```

## Markup

```html
<!-- Column -->
<div class="container">
  <div class="row">
    <div class="col-md-4"></div>
    <div class="col-md-4"></div>
    <div class="col-md-4"></div>
  </div>
</div>
```
